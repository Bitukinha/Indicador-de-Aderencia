# Indicador de Aderência

Painel para acompanhamento de pedidos e atrasos.

## Como rodar localmente

Requisitos: Node.js e npm/yarn. Depois:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Este projeto pode ser implantado no Vercel. Veja as instruções no README ou peça que eu gere a configuração de deploy.
